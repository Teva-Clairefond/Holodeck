# JOB04 - Configuration réseau de la VM cliente


## Explication : 

La VM doit être sur le même réseau dit "L2" (de couche 2 du modèle OSI) avec une interface réseau up. Ici l'interface réseau est dans le même LAN, que nous avons définit lors de la création de nos VMs.
